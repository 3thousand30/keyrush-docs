# Key Rush Docs

Hosted documentation for [Key Rush](https://apps.microsoft.com/detail/9MW7722B1026), the desktop typing game by [3thousand30](https://3thousand30.com/).

## What lives here

- product home / install link
- getting started guide
- keyboard controls
- privacy policy
- support links

## Public URLs

- Docs: <https://3thousand30.github.io/keyrush-docs/>
- Microsoft Store: <https://apps.microsoft.com/detail/9MW7722B1026>

## Notes

- The app is local-first and offline-first.
- Share actions can save a local PNG and open external composers only when the player chooses.
- This repo is intentionally small and simple, modeled after the 3thousand30 desktop docs sites.
