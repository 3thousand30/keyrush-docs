# Getting Started

## Install

Download **Key Rush** from the [Microsoft Store](https://apps.microsoft.com/detail/9MW7722B1026).

Once installed, launch it from the Start menu or your app list.

## First launch

On the first run, Key Rush asks for a local player nickname before you enter the main menu.

That nickname is used for:

- local leaderboards
- run history
- settings and best-score context

No account is required.

## Main modes

### Story

Story mode is the mission-driven path.

- 3 chapters
- 3 levels per chapter
- countdown pressure
- chapter-specific hero art and themed code screens

### Endless

Endless mode is the falling-word pressure run.

- starts slower
- ramps on a fixed curve
- ends when a falling word is missed

### Zen

Zen mode is the no-fail passage mode.

- a bundled passage is chosen at random
- one target word is highlighted at a time
- each correct word rotates to a new target
- you can stop whenever you like

## Results and sharing

Result screens can:

- save a PNG result card to `Downloads/KeyRush/`
- copy share text and image when supported by your desktop
- open external composers for X, Threads, Facebook, LinkedIn, Reddit, and WhatsApp

These actions happen only when you choose them.
