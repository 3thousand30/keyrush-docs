# Keyboard Controls

Key Rush is designed to be fully usable from the keyboard.

## Global

- `Tab` / `Shift+Tab` move focus
- `Enter` / `Space` activate the focused button
- `Escape` closes the open menu or modal
- `F1` or `?` opens Help

## Home / Menus

- `P` opens the player switcher
- `S` opens Settings
- `Escape` closes the player or feedback menu if open

## Gameplay

- Type the active target word
- Press `Enter` to submit your current input
- In **Endless** and **Zen**, type `stop` and press `Enter` to end the run

## Modal behavior

- Opening **Settings** or **Players** keeps the current screen in place
- `Escape` or clicking the backdrop closes the modal
- Focus returns to the button that opened it
